# Circus-Of-Plates-Game
A game in which the player should catch falling pokeballs and plates.
Every 3 catches with the same color increases the score.

Implemented 10 different design patterns:
1. Singleton
2. Dynamic Linkage
3. Abstract Factory
4. Factory
5. Fly Weight
6. Memento
7. Iterator
8. Observer
9. Strategy
10. State
